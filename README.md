# WebSocket Satochi/Tom


## Requisites

- python >= 3.9
- pip >= 22.0.3


## Installation

```sh
git clone https://github.com/stalinscj/websocket-satochi.git
```

```sh
cd websocket-satochi
```

```sh
pip install -r requirements.txt
```

```sh
python -m pip install pytest
```


## Running tests

```sh
pytest
```


## Running script

```sh
python main.py
```
